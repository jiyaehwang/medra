# 섹션 노출할 정보 노출 방식 비고
— NAV (고정) 로고, 섹션 앵커 링크 (기능·워크플로우·피드백), 베타 시작 버튼 상단 고정. 스크롤 시 반투명 블러 처리. 우측에 CTA 버튼 상시 노출 CTA
Z패턴 기준 우상단 CTA 위치. 스크롤과 무관하게 서비스 입장 버튼이 항상 보여야 함. 문구는 "베타 시작하기"처럼 즉시성이 담긴 동사 사용
1 HERO 서비스 한줄 가치 제안, 대상 사용자 명시, CTA 버튼 2개, 터미널 데모 카드 2컬럼. 좌측 카피+CTA(Z패턴 좌상단→좌하단), 우측 터미널 목업 카드(결과 이미지) CTA 카피
첫 화면 안에서 "무슨 서비스인지"가 완결되어야 함. 헤드라인은 추상어 지양 — "의료규제, 놓치는 것 없이 파악하세요"처럼 구체적 결과를 먼저 말하기. CTA 주변에 "클로즈드 베타 · 무료" 마이크로 카피 배치 권장
2 베타 안내 배너 클로즈드 베타 성격 설명, 베타 기간 (2026.05.11~11.30), 참여 버튼 다크 그린 풀-와이드 배너. 좌측 카피, 우측 날짜 카드 + 참여 버튼 CTA
히어로 직후 CTA 재노출 구간. "지금 무료로 쓸 수 있다"는 즉시성·희소성 전달. 긴 페이지 기준 CTA는 3~5회 반복이 표준이며 이 배너가 2번째 CTA 역할
3 핵심 기능 Agent Swarm, 종합보고서 자동 작성, 자연어 입력, 대화 맥락 유지 2×2 그리드 카드. 넘버링 + SVG 아이콘 + 제목 + 설명. 벤토 그리드 구조 레이아웃 카피
기능 나열이 아닌 결과 중심 카피 원칙 적용 — "자연어 입력"보다 "명령어 없이 바로 질문"처럼 사용자가 얻는 결과로 서술. 벤토 그리드로 정보가 많아도 시각적 정돈감 유지
4 이런 분들에게 추천 직군별 기존 어려움(Pain) + Medra 해결 방식(Solution) + 키워드 칩 3컬럼 카드. SVG 아이콘, Pain 강조 박스(좌측 골드 라인), Solution 설명, 하단 키워드 칩 신뢰 카피
"나한테 맞아?" 질문에 답하는 구간. 방문자가 자기 직군을 즉시 발견하도록 설계. Pain 박스는 공감 → Solution으로 이어지는 스토리 구조. 수치 보강 시 신뢰도 상승 ("IND 제출 평균 4시간 → 15분" 등)
5 도입 전·후 비교 기존 방식 비효율 3가지 vs Medra 사용 후 개선 3가지 + 소요 시간 비교 2컬럼 카드. Before(밝은 배경) / After(다크 배경). 하단 소요 시간 강조 행 신뢰
큰 숫자 시각화 원칙 적용 — "2~4시간 → 5~10분" 수치를 텍스트가 아닌 강조 타이포로 처리 권장. 사회적 증거 구간으로 "얼마나 유용한가?"에 수치로 답해야 설득력이 생김
6 직군별 활용 예시 직군별 실제 Q&A 시나리오 각 2개 (임상·의약품·의료기기) 탭 UI. 탭 클릭 시 해당 직군 예시만 노출. 다크 배경 채팅 버블 형식 신뢰 레이아웃
Accordion/탭 방식으로 페이지 길이 억제하면서 정보 밀도 유지. 실제 업무 언어 기반 시나리오가 핵심 — 추상적 설명보다 "이런 질문에 이런 답이 나온다"는 구체적 결과 이미지 제시
7 권장 사용 순서 웹 검색 → 규제 검색 → 종합보고서 작성 3단계 가로 3단계 플로우. 원형 넘버링 + 단계별 설명 레이아웃
진입 장벽 제거 구간. "어떻게 쓰는 거야?"에 한 화면에서 답해야 함. 한 화면 하나의 메시지 원칙 — 단계별로 시선이 자연스럽게 흐르도록 여백 충분히 확보
8 피드백 루프 @feedback 커맨드, 즉시 전달 방식, 24시간 반영 약속, 공동 설계 메시지 다크 풀-와이드 섹션. 대형 헤드라인 + @feedback 커맨드 박스 + 특징 카드 3개 신뢰 카피
베타 서비스 핵심 차별점. "베타인데 괜찮아?"라는 불안을 해소하는 신뢰 구간. 거대 타이포그래피로 @feedback 커맨드 자체를 시각적 앵커로 활용. "24시간 반영" 같은 즉시성 문구가 마이크로 카피 역할
9 시작하는 법 4단계 접속 가이드 + 실제 입력 가능한 회원가입 폼 2컬럼. 좌측 단계별 가이드, 우측 회원가입 폼 (이메일/비밀번호/확인, 실시간 검증) CTA
최종 행동 유도 구간. 폼 자체가 CTA. "무료로 시작하기" 버튼 문구에 즉시성 유지. 폼 하단에 "이미 계정이 있으신가요? 로그인" 마이크로 카피로 이탈 방지. 단계 가이드는 "시작이 어렵지 않다"는 심리적 안전감 제공
— FOOTER 로고, 베타 기간, 서비스 소개 텍스트 1행 가로 정렬. 현재 최소 구성 신뢰
현재 법적 고지 항목 부재 — 문의 이메일, 이용약관, 개인정보처리방침 추가 필요. 3차 요소(부가 정보)이나 신뢰도에 직접 영향. 다크 배경 2컬럼으로 개선 권장
